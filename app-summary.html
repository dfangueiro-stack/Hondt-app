Classic Snake

One-page repo summary based only on files in this repository

Static Browser App

What It Is

Classic Snake is a minimal browser-based Snake game with classic rules only. The repo provides a lightweight static page, styling, and one browser-safe script that contains both game logic and rendering/input wiring.

Who It's For

Primary persona: players who want a simple browser-playable Snake game with no framework or install-heavy setup. Developer persona: someone who wants a small static web app that is easy to open and inspect.

What It Does

  • Renders a 20x20 game board in the browser.
  • Starts movement from arrow keys or WASD.
  • Prevents instant reversal into the snake's opposite direction.
  • Spawns food on unoccupied cells and grows the snake on eat.
  • Tracks score and updates live status text in the UI.
  • Ends the game on wall collision or self-collision.
  • Supports pause/resume, restart, and on-screen direction buttons.

How It Works

UI shell

index.html defines the page shell, scoreboard, status, action buttons, board container, and on-screen controls.

Presentation layer

styles.css styles the panel layout, responsive behavior, board grid, and cell states for snake head/body and food.

Game engine + controller

src/game.js holds state creation, direction changes, pause/restart, tick progression, collision checks, food spawning, and status generation.

Data flow

User input (keyboard/buttons) updates in-memory state, a setInterval tick advances the game every 140 ms, and render logic maps state back onto DOM cells, score, and status text.

Services / persistence

Not found in repo: no backend, database, API calls, local storage, build system, or framework runtime.

How To Run

  1. Open index.html directly in a browser.
  2. Optional: serve the folder with any static server, such as npx serve . or python -m http.server.
  3. Load the page, then use arrow keys or WASD to begin.
